I designed this website as a part of my English 235 class assignment to design a blog. I had wanted to make a Resume/Blog/E-Portfolio style website in the past and I figured this was the perfect opportunity to. This website is made in HTML, CSS, and JavaScript and is hosted for free using GitHub pages.

The purpose of this assignment was to to use skills learned in this course to create an online document that showcases written and visual organization, design, and style. This project also enhances how to research, plan, and partition a subject with the purpose to inform and persuade. Lastly, this project highlights my achievements as a student and showcases documents designed in this and other courses.
