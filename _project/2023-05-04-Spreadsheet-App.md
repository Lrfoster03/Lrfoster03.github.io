I created this project for my CPT_S 321 Object Oriented Programming course at WSU. The goal of the project was to create a spreadsheet application that could perform basic calculations and data manipulation, as well as learning the basics of object-oriented programming. The project was a great way to learn about classes, inheritance, and polymorphism in Python. It was built using .NET 8.0 and Windows Forms.
